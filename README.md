# Flight Fight

Flight Fight is a 3D board game prototype developed in Unity.

This project was created for the module CMP-6056B / CMP-7042B at the University of East Anglia.

## Game Overview

Flight Fight is inspired by traditional aeroplane chess games. Players move their aircraft token along a board path based on dice rolls.

The goal of the game is to reach the end of the board by rolling the dice and moving forward.

## Implemented Features

The prototype currently includes several working gameplay elements:

- Dice rolling system
- Aircraft token movement
- User interface displaying dice results
- 3D board environment

## Technologies Used

- Unity 3D
- C#
- TextMeshPro

Future features planned for the game include:
